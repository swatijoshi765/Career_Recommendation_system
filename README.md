# Career_Recommendation_system


This is a Streamlit-based machine learning project that recommends the best career based on user interests.

How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

Features
- ML-based career recommendations
- Top 3 matches with confidence score
- Roadmaps, courses, skills, salary info
- Beautiful UI using Streamlit + CSS
